Exemple d’opacité et superposition

Cette boîte est semi-transparente (fond seulement).

Comparaison des comportements

Opacity 0.4
Visibility hidden
Display none

→ La boîte rouge est visible et translucide.
→ La boîte verte existe mais est invisible.
→ La boîte orange est complètement retirée.
→ Vous pouvez modifier les z-index pour changer l’ordre des superpositions.
