Caractéristiques des Éléments de Type Bloc et En Ligne

Éléments de Type Bloc

Chaque élément de type bloc commence sur une nouvelle ligne et occupe toute la largeur de son conteneur parent.

Exemples d'éléments blocs : <div>, <p>, <h1> à <h6>, <ul>, <table>.

  • Utilisation : Structurer les sections de contenu sur une page.
  • Propriétés appliquées : width, height, margin, padding.

Éléments de Type En Ligne

Les éléments en ligne restent dans le flux du texte et n'occupent que la largeur de leur contenu. Ils se placent côte à côte sans retour à la ligne. Exemples d'éléments en ligne : <span>, un lien (<a>), texte en gras (<strong>).

Les éléments en ligne sont parfaits pour styliser du texte ou des petites images sans perturber la structure.
