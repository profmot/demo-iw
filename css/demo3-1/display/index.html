Élément avec display: inline-block

Description :

  • inline : reste dans le flux du texte et n’occupe que l’espace de son contenu.
  • block : occupe toute la largeur disponible et commence sur une nouvelle ligne.
  • none : n’est pas affiché et ne prend aucune place.
  • inline-block : se comporte comme un élément en ligne mais peut être redimensionné.
